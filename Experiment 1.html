EXPERIMENT 1: Characterization of Maximal Bicliques using the Consensus Algorithm¶

In this Experiment we run the implementation of maximal biclique enumeration algorithm from Alexe et al. (http://citeseer.nj.nec.com/alexe02consensus.html)

biclique2.h: The implementation of the algorithm named Consensus Algorithm.

sbtest_ii.cpp: The main program to handle the file input/output.

To compile it (under Linux or Mac OSX), simply use the command:
$ g++ -o sbtest sbtest_ii.cpp

To run it, use the command line (all the file names are mandatory):
$ sbtest [input bigraph file] [output bclique file] [output size file]

A bipartite graph has two node sets, say X and Y. Each edge is represented by a node x from X and a node y from Y.

The input file consists of a list of edges where x is in the first column and y is in the second column.
For example:
2 2
2 3
2 4
3 1
3 2
3 3

In the output file, each biclique is represented by 3 lines:
[list of nodes from 1st node set]
[list of nodes from 2nd node set]
[empty line]
For example:
3
1 2 3

2 3
2 3

2
2 3 4

class Spice:
    def __init__(self, id, name):
        self.id = id
        self.name = name


class Disease:
    def __init__(self, id, name):
        self.id = id
        self.name = name


my_file = open('level-3-asssociations.tsv','r')
lines = my_file.readlines()

spices = []
diseases = []
for line in lines:
    temp_list = line.split('\t')
    spices.append(temp_list[1])
    diseases.append(temp_list[3])

# spices.pop(0)
# diseases.pop(0)
spices = list(set(spices))
diseases = list(set(diseases))

disease_mapped_list = []
i = 0
for d in diseases:
    disease = Disease(i, d)
    disease_mapped_list.append(disease)
    i += 1

spice_mapped_list = []
i = len(diseases)
for s in spices:
    spice = Spice(i, s)
    spice_mapped_list.append(spice)
    i += 1

j = 0
for s in spice_mapped_list:
    if j == 0:
        my_spice_file = open('Spice.txt', 'w')
        my_spice_file.write(str(s.id) + ' ' + s.name + '\n')
        j += 1
    else:
        my_spice_file = open('Spice.txt', 'a')
        my_spice_file.write(str(s.id) + ' ' + s.name + '\n')
j = 0
for d in disease_mapped_list:
    if j == 0:
        my_spice_file = open('Disease.txt', 'w')
        my_spice_file.write(str(d.id) + ' ' + d.name + '\n')
        j += 1
    else:
        my_spice_file = open('Disease.txt', 'a')
        my_spice_file.write(str(d.id) + ' ' + d.name + '\n')

edge = []
for line in lines:
    temp_list = line.split('\t')
    spice_name = temp_list[1]
    disease_name = temp_list[3]
    if spice_name == 'Spice' and disease_name == 'Disease':
        continue
    for s in spice_mapped_list:
        if s.name == spice_name:
            spice_id = s.id
    for d in disease_mapped_list:
        if d.name == disease_name:
            disease_id = d.id
    el = []
    el.append(spice_id)
    el.append(disease_id)
    edge.append(el)

j = 0
for e in edge:
    if j == 0:
        my_edge_file = open('Edge-negative.txt', 'w')
        my_edge_file.write(str(e[0]) + ' ' + str(e[1]) + '\n')
        j += 1
    else:
        my_edge_file = open('Edge-negative.txt', 'a')
        my_edge_file.write(str(e[0]) + ' ' + str(e[1]) + '\n')

    temp_list = line.split(' ', 1)
    diseases[temp_list[0].strip()] = temp_list[1].strip()

biclique = []

controller = 1
f = open('biclique.out', 'r')
lines = f.readlines()
for line in lines:
    s = []
    d = []
    if controller == 1:
        temp = line.split()
        if len(temp) == 1:
            s.append(spices[temp[0]])
        else:
            temp = line.split()
            for t in temp:
                s.append(spices[t])
        biclique.append(s)
        controller = 2
    elif controller == 2:
        temp = line.split()
        if len(temp) == 1:
            d.append(diseases[temp[0]])
        else:
            temp = line.split()
            for t in temp:
                d.append(diseases[t])
        biclique.append(d)
        controller = 3
    else:
        controller = 1

Case 1:¶

Analysis of the biclique size(Number of spices + Number of diseases):¶

Cumulative distribution of the biclique size was obtained as shown and the results were compared with the histogram of frequency of spice and diseases. The inferences were in complete agreement with each other, with the consensus that the bicliques obtained with very high and very small biclique size are very few in number with most of the bicliques having average sizes. For instance, the percentage of bicliques having biclique size from 0 to 15 are just 4% of the total number of bicliques. Also, the percentage of bicliques having biclique size from 50 to 100 are just 0.5% of the total bicliques as the percentage of bicliques rose from approximately 75% to 75.5%.

Case 3:¶

Spice-Disease relationship analysis:¶

The plot of the number of spices against the number of diseases for every biclique obtained follows the power law. This is evident from the plot as shown below. There exists very few bicliques with very high spice count along with very low disease count and with very low spice count along with very high disease count. Most of the bicliques do not exhibit this sort of a polar behaviour. The histograms of the spice count and the disease count also infuse confidence in the argument presented above as it mimics a normal distribution. Furthermore, cumulative distribution was obtained for spice count and disease count against the fraction of bicliques and again the results were in complete agreement. Hence, this argument stands validated.
